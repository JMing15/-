## Bit Manipulation
* 异或的特性。第136题。
```
x ^ 0 = x（136题）
x ^ x = 0（136题） 
a ^ b ^ c = a ^ (b ^ c) = (a ^ b) ^ c（结合律）（136题）
```

* 有特殊意义的&位操作运算
```
x &= (x-1) 将最低位（LSB）的1清零 (201题、231题)
x & -x 得到最低位(LSB)的1
```
